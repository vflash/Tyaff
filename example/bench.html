🛠 Сравнение memo API

Как предотвратить лишний рендер дочернего компонента, когда поменялись только props родителя

tyaff

const Item = Component({
  memo(p) {
    return [p.value];
  },
  render(p) {
    return h('div', null, p.value);
  }
});
Функция в определении + deps-массив

React

const Item = React.memo(
  function Item({ value }) {
    return h('div', null, value);
  }
);
HOC-обёртка + shallowEqual по умолчанию

Preact

const Item = memo(
  function Item({ value }) {
    return h('div', null, value);
  },
  (prev, next) =>
    prev.value === next.value
);
HOC-обёртка + ручная функция сравнения
tyaff: memo — это просто функция в объекте определения компонента. Возвращает deps-массив (как useEffect). Нет HOC, нет компаратора, нет обёрток. Компонент остаётся одним целым.
